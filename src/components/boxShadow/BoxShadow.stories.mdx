# Box shadow

With `MDX` we can define a story for `Box shadow` right in the middle of our markdown documentation.

## Default

<Story name="default">
  <Container>
    {Object.keys(DP_TYPES).map(dp => (
      <Wrapper>
        <ColoredBox backgroundColor="#666666" boxShadow={DP_TYPES[dp]} />
        <Label color="#cccccc">{dp.replace("_", " ").toLowerCase()}</Label>
      </Wrapper>
    ))}
  </Container>
</Story>
