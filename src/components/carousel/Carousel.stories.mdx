# Carousel

With `MDX` we can define a story for `Carousel` right in the middle of our markdown documentation.

## Default

<Story name="default">
  <Wrapper>
    <Carousel
      interval={null}
      slides={[
        <Item>First slide</Item>,
        <Item>Second slide</Item>,
        <Item>Third slide</Item>
      ]}
    />
  </Wrapper>
</Story>
