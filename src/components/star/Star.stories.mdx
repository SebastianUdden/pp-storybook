# Star

With `MDX` we can define a story for `Star` right in the middle of our markdown documentation.

## Default - not visible

<Story name="default">
  <Star
    id="Default"
    color={"orange"}
    remainderRatio={0}
    {...attributes}
    {...actions}
    onClick={() => {}}
  />
</Story>

## Dynamic

<Story name="dynamic">
  {() => {
    const [color, setColor] = useState("grey");
    return (
      <Star
        id="Dynamic"
        color={color}
        remainderRatio={1}
        {...attributes}
        {...actions}
        onClick={() => setColor(color === "grey" ? "orange" : "grey")}
      />
    );
  }}
</Story>
